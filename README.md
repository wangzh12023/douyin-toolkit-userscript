# Douyin Toolkit Userscript

一个用于抖音网页版的 Tampermonkey 脚本，支持提取已加载作品链接、提取作品名称和链接，并批量取消喜欢/收藏。

## 功能

- 提取当前页面已加载的视频和图文作品链接
- 提取“作品名称 + 链接”，适合粘贴到表格
- 选择作品后批量取消喜欢
- 选择作品后批量取消收藏
- 批量任务可停止，取消喜欢/收藏会尽量使用网页接口
- 支持自动滚动加载更多已显示内容

## 安装

1. 安装 Tampermonkey 或 Violentmonkey。
2. 新建脚本。
3. 将 `douyin-url-extractor.user.js` 的内容复制进去并保存。
4. 打开 `https://www.douyin.com/`，页面右侧会出现“抖”按钮。

## 使用

- `提取作品链接`：复制当前页面已经加载到 DOM 中的视频/图文链接。
- `提取作品链接+名称`：复制标题和链接，每行一条。
- `批量取消喜欢`：先勾选作品，再通过接口批量处理。
- `批量取消收藏`：先勾选作品，再通过接口批量处理。
- `停止批量任务`：中断新标签页中的点击兜底任务。
- `修改脚本设置`：开启自动滚动、复制前选择、保留链接参数等。

## 说明

脚本只处理当前账号在网页端可见、已加载的内容。抖音网页接口和页面结构可能调整；如果批量取消功能失效，需要重新抓取真实网页请求后更新脚本。
